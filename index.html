Kentucky Salamander Heat Map

Map authored by: Jill Newman

This map illustrates salamander species occurrence data in Kentucky using citizen science data. Species data was obtained from Global Biodiversity Information Facility (GBIF). This map was created using QGIS. More information on GBIF is available from the Global Biodiversity Information Facility page. This page was created by Jill Newman for New Maps Plus.

source: GBIF.
